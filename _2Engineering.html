Omni Directional Antenna Array System

The array system reduce transmission power requirements, increase connectivity and reduce signal noise. Our team designed and printed antenna array and phase delay network. We designed and printed RF Receiver and Signal Strength Detector. We implemented control phase delays through Arduino micro-controller by detecting optimum signal strength

Altera DE2 traffic lights

We used a Altera DE2 board and implemented a traffic light system. The lights will automatically loop if there were no requests made. If there was a request made via flipping a switch it will prioritise to address that service call.

Ball on beam

For one of my controls courses, my and a friend created a LabView project that would always maintain the position of the ball at the middle of the beam. The system was tested by nudging the ball, momentarily disturbing the beam and adding weight to either side of the beam.
